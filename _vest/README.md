# vitest



## インストール



### テストの雛形サイトを作る

`pnpm create vite`

色々選択する

```
Select a framework:
->Vue
Select a variant:
->JavaScript
```



### テストをインストール

`pnpm add -D vitest`



追記する

```
 "scripts": {
    "test":"vitest"
 }
```



テスト実行

`pnpm test`
